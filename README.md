# 🛒 Maverik Store - Fullstack E-Commerce

Chào mừng đến với Repo của dự án **Maverik Store** (Nhóm 39). Đây là nền tảng E-commerce chuyên thời trang Streetwear mang phong cách UI/UX từ maverikstudio.vn.

Dự án được phân tách 2 tầng: **Frontend** (Vite + VanillaJS + Bootstrap 5) và **Backend** (Node.js + Express + Prisma + MySQL).
